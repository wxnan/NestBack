## 应用简介

### 什么是归巢？

归巢是一款专注于家庭收纳与物品管理的应用，帮助你轻松管理家中物品的存放位置、过期时间、库存数量等信息。

### 核心特性

- **本地优先**：所有数据存储在本地设备，离线可用
- **智能提醒**：自动提醒即将过期或库存不足的物品
- **空间层级**：支持多层级空间管理，模拟真实收纳结构
- **扫码录入**：支持条形码扫描快速添加物品
- **数据统计**：多维度统计分析，助你了解物品分布
- **灵活扩展**：支持AI识图、扫码API、WebDAV备份等高级功能

## 快速开始

### 1. 首次使用

首次打开应用后，默认会创建一个"我的家"家庭空间。

### 2. 添加第一个空间

1. 切换到"空间"标签页
2. 点击右下角的 **+** 按钮
3. 输入空间名称（如"客厅"、"厨房"、"冰箱"等）
4. 选择一个图标
5. 点击保存

### 3. 添加第一个物品

**方法一：手动录入**

1. 切换到"录入"标签页
2. 点击"手动录入"
3. 填写物品基本信息（名称、分类、位置、数量等）
4. 点击保存

**方法二：扫码录入**

1. 切换到"录入"标签页
2. 点击"扫码录入"
3. 对准商品条形码扫描
4. 系统会自动填充物品信息
5. 确认信息后点击保存

**方法三：滑动手势快速操作**

1. 在首页找到要操作的物品
2. 向右滑动：显露 \[-1] 和 \[+1] 按钮，快速调整数量
3. 向左滑动：显露 \[复制] 和 \[拆分] 按钮

## 首页功能详解

### 顶部导航

- **家庭切换器**：点击左上角家庭名称，可在多个家庭间快速切换
- **搜索栏**：支持物品名称的搜索

### 统计卡片

- **已过期**：红色卡片，点击查看所有已过期物品
- **即将过期**：黄色卡片，点击查看30天内即将过期的物品（可在提醒设置中调整天数，默认30天）

### 筛选功能

- **分类筛选**：横向滚动芯片组，按物品分类筛选
- **标签筛选**：横向滚动芯片组，按物品标签筛选（无标签时隐藏）

### 物品列表

- **排序切换**：点击右上角图标，切换排序方式
  - 最近添加
  - 最近过期
  - 最近修改
- **物品卡片显示信息**：
  - 左侧：物品缩略图（点击可放大预览图片）
  - 中间：物品名称、存放位置、当前数量
  - 右侧：距离过期/过保的天数
- **滑动操作**：
  | 滑动方向 | 操作按钮     | 说明           |
  | ---- | -------- | ------------ |
  | 向右滑  | \[-1] 消耗 | 数量减1         |
  | 向右滑  | \[+1] 补货 | 数量加1         |
  | 向右滑  | \[扔掉]    | 已过期物品直接移至垃圾桶 |
  | 向左滑  | \[复制]    | 复制物品信息创建新物品  |
  | 向左滑  | \[拆分]    | 将物品拆分为多个     |
- **长按操作**：进入多选模式，可批量移动或删除物品

## 空间管理

### 空间层级

支持无限层级的空间嵌套，模拟真实收纳结构：

```
我的家
├── 客厅
│   ├── 电视柜
│   │   ├── 第一层抽屉
│   │   └── 第二层抽屉
│   └── 茶几
├── 厨房
│   ├── 冰箱
│   │   ├── 冷藏室
│   │   └── 冷冻室
│   └── 橱柜
└── 卧室
    └── 衣柜
```

### 特殊空间

每个家庭都有两个不可删除的特殊空间：

- **待整理**：用于临时存放位置还未确定的物品
- **回收站**：用于存放已删除但可能需要恢复的物品
- 回收站-垃圾桶：用于存放扔掉的过期物品

### 添加子空间

1. 进入要添加子空间的位置
2. 点击右下角 **+** 按钮
3. 选择"添加空间"
4. 填写空间名称和图标
5. 点击保存

### 移动物品

1. 长按物品进入多选模式
2. 选择要移动的物品
3. 点击顶部"移动"按钮
4. 在弹出的位置选择器中选择新位置
5. 点击确认

### 回收站操作

进入回收站后：

- **左滑物品**：显露 \[删除]（永久删除）和 \[恢复]（编辑后恢复）按钮

## 物品录入

### 录入方式总览

| 方式   | 说明        | 配置要求         |
| ---- | --------- | ------------ |
| 手动录入 | 直接填写物品信息  | 无            |
| 扫码录入 | 扫描商品条形码   | 需配置扫码API（可选） |
| AI识图 | 拍照自动识别物品  | 需配置AI API    |
| AI聊天 | 语音/文字描述物品 | 需配置AI API    |

### 手动录入

1. 点击"录入"标签页
2. 点击"手动录入"
3. 填写物品信息：
   - **基本信息**（必填）
     - 封面图片（可选）
     - 名称（必填）
     - 分类（必填）
     - 二级分类（可选）
     - 存放位置（必填）
     - 数量（默认1）
     - 单位（件、个、瓶等）
     - 最低库存提醒（可选）
     - 单价（可选）
     - 总价（可选，可根据数量和单价自动计算）
     - 标签（可选）
     - 备注（可选）
   - **扩展信息**（根据分类显示不同字段）
     - 品牌、厂商、规格、颜色
     - 生产日期、保质期、过期日期
     - 购买日期、保修期、过保日期
     - 开封日期
     - 条形码
     - 等等
4. 点击保存

### 扫码录入

1. 点击"录入"标签页
2. 点击"扫码录入"
3. 对准商品条形码
4. 系统调用配置的扫码API获取商品信息
5. 自动填充表单（名称、图片等）
6. 补充其他信息后保存

> 💡 **提示**：如果未配置扫码API，系统会提示配置或跳转到手动录入模式。

### 物品复制

1. 在首页或空间中找到要复制的物品
2. 向左滑动，点击"复制"
3. 进入复制物品详情页
4. 修改名称、数量、位置等信息
5. 点击保存

### 物品拆分

1. 在首页或空间中找到要拆分的物品（数量需>1）
2. 向左滑动，点击"拆分"
3. 进入拆分物品详情页
4. 填写新物品的数量（需小于原物品数量）
5. 补充其他信息后点击保存
6. 原物品数量会自动减少

### 物品补货

1. 在回收站、垃圾桶空间中，或在已经过期、即将过期物品列表中找到要补货的物品
2. 向左滑动，点击"补货"
3. 进入补货物品详情页
4. 填写新物品的名称、数量、位置等信息
5. 点击保存

## 统计中心

### 核心指标卡片

| 指标   | 说明           |
| ---- | ------------ |
| 物品总数 | 当前家庭的所有物品数量  |
| 物品总价 | 所有物品的总价值     |
| 已经过期  | 已超过保质期的物品数   |
| 即将过期 | 30天内即将过期的物品数  |
| 库存不足 | 数量低于设定阈值的物品数 |
| 等待估值 | 尚未设置价格的物品数   |

### 统计饼图

支持多维度统计：

- **按分类**：查看各分类物品分布
- **按房间**：查看各空间物品分布
- **按标签**：查看各标签物品分布
- **按成员**：查看各创建者的物品分布

### 下钻交互

1. 点击饼图的某个扇区
2. 可以继续在该分类下按其他维度筛选
3. 筛选结果直接显示在饼图下方

## 高级配置

### WebDAV备份配置

WebDAV可以将本地数据备份到云盘（如坚果云、阿里云盘等）。

**配置步骤**：

1. 进入"我的" > "数据管理" > "WebDAV设置"
2. 填写以下信息：
   - 服务器地址：如 `https://dav.jianguoyun.com/dav/`
   - 用户名：你的云盘账号
   - 密码：你的云盘密码
   - 备份路径：如 `/NestBack/`
   - 加密密钥：（可选）用于加密备份数据
3. 点击"测试连通性"
4. 测试成功后，点击"立即备份"

### Supabase云同步配置

Supabase提供实时数据同步功能，支持多设备同步和多成员协同。

**配置步骤**：

1. 进入"我的" > "数据管理" > "Supabase设置"
2. 填写以下信息：
   - Supabase URL：从Supabase项目获取
   - API Key：从Supabase项目获取
3. 点击"测试连通性"
4. 复制提供的SQL脚本到Supabase控制台执行
5. 完成后，可通过"邀请成员"生成邀请码

### AI功能配置

AI功能包括AI识图（拍照识别物品）和AI聊天（语音/文字描述物品）。

**配置步骤**：

1. 进入"我的" > "数据管理" > "AI设置"
2. 选择配置模式：
   - **家庭公用**：管理员配置，所有成员共用
   - **个人私有**：成员自行配置，优先级更高
3. 填写以下信息：
   - Base URL：AI服务地址（如OpenAI、Ollama等）
   - API Key：访问密钥
   - Vision模型：用于识图的模型
   - Chat模型：用于聊天的模型
4. 点击"测试连通性"和"测试余额"

> ⚠️ **注意**：AI服务会产生费用，请确保账户余额充足。

### 扫码API配置

扫码API用于查询商品条形码对应的商品信息。

**配置步骤**：

1. 进入"我的" > "数据管理" > "扫码设置"
2. 填写API密钥
3. 点击"测试条码查询"

## 常见问题

### Q: 如何找回误删的物品？

A: 进入"空间" > "回收站"，找到要恢复的物品，向左滑动点击"恢复"即可。

### Q: 为什么扫码录入没有自动填充商品信息？

A: 请检查是否已配置扫码API。配置路径："我的" > "数据管理" > "扫码设置"。如果未配置，系统会提示配置或跳转到手动录入。

- 条码查询失败：请检查API密钥是否正确。
- 扫码录入失败：请检查网络连接是否正常。
- 条码查询未找到：说明该条码商品信息未在数据库中。请更换API提供商尝试。
- 填充部分商品信息；可能是数据库中缺少该条码的详细商品信息。请更换API提供商尝试。

### Q: 如何调整过期提醒的时间？

A: 进入"我的" > "物品管理" > "提醒设置"，可以设置：

- 提醒时间（默认09:00）
- 过期提前提醒天数（默认30天）
- 低库存阈值（默认1件）

### Q: 物品数量为0时会被自动删除吗？

A: 不会。当物品数量减至0时，系统会自动将其移至"回收站"，而不是直接删除。你可以在回收站中恢复或永久删除。

### Q: 如何在不同设备间同步数据？

A: 需要配置Supabase云同步服务。配置路径："我的" > "数据管理" > "Supabase设置"。配置完成后，同一家庭的设备可以实时同步数据。

### Q: 本地数据可以备份吗？

A: 可以。通过WebDAV功能将数据备份到云盘。配置路径："我的" > "数据管理" > "WebDAV设置"。

### Q: 如何创建多个家庭？

A: 进入"我的" > "账户与家庭" > "家庭管理"，点击"创建新家庭"即可。

### Q: 分类可以自定义吗？

A: 可以。进入"我的" > "物品管理" > "分类管理"，可以添加、编辑、删除分类。

### Q: 如何使用AI功能？

A: 需要先配置AI服务。配置路径："我的" > "数据管理" > "AI设置"。配置完成后，在"录入"页面可以使用"AI识图"和"AI聊天"功能。

### Q: 应用支持离线使用吗？

A: 支持。所有基础功能（手动录入、物品管理、统计等）都可以在离线状态下正常使用。AI识图、扫码API、云同步等功能需要网络连接。
