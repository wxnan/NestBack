## 🆘 紧急情况

### 数据丢失怎么办？

1. **有备份**：进入"我的" > "数据管理" > "WebDAV设置" > "恢复备份"
2. **无备份但有云同步**：检查Supabase中是否有数据
3. **完全无备份**：联系技术支持

### 应用无法使用？

1. 重启应用
2. 重启设备
3. 清除应用缓存
4. 卸载重装应用
5. 联系技术支持
