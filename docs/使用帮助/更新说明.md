# v1.1.1
修复并优化多家庭数据共享与WebDAV备份功能

1. 重构分类、属性、标签为全局共享，移除家庭绑定逻辑并添加数据库迁移
2. 新增物品跨家庭移动功能
3. 完善WebDAV备份服务：支持按家庭名称备份、恢复与跨家庭恢复
4. 修复多处Provider加载逻辑，移除houseId参数传递
5. 添加备份设置页面多家庭恢复功能
6. 优化搜索与空间卡片拖拽排序功能

注意：
- 请在更新前通过WebDAV备份服务或导入导出功能备份重要数据，避免丢失。
- 此版本需要卸载旧版本，以应用新的数据库结构。
